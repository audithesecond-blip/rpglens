Pay for what you use. Larger programs consume more RLUs — smaller ones cost less. No fixed analysis limits. No credit card required to start.

How RPGLens Units (RLUs) work: Every action consumes RLUs based on program size — not a fixed count. Analysis = 1 RLU per 100 lines per tab. Conversion = 5 RLUs per 100 lines. A 1,000-line program analysed across all 5 tabs = 50 RLUs. A 5,000-line program = 250 RLUs. Unused RLUs roll over at 50% on renewal. Top-up packs available any time.
